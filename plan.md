# 8 bit processor

## Memory 
1. ROM
    - 0x0000 - 0x1FFF
2. RAM
    - 0x2000 - 0x4FFF
3. STACK
    - 0x5000 - 0x5FFF
4. I/O
    -0x6000 - 0x6FFF
## Registers
1. Accumulator(A)
    - size: 8 bits
2. Program Counter(PC)
    - size: 16 bits
3. Stack Pointer(SP)
    - size: 16 bits
4. General Purpose Registers(GPRs)
    - number: ?
    - size: 8 bits
5. Flags Register(F)
    - size: 8 bits
6. Instruction Register(IR)
    - size: 8 bits
8. Memory Address Register(MAR)
    - size: 16 bits
9. Memory Data Register(MDR)
    - size: 8 bits
10. Base Pointer(BP)
    - size:16 bits
